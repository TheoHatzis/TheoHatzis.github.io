CV Builder

 

I like using Python automation for basic things.

 

For CV building MSWord you can use the help of Ninja2 with docxtpl templates together with a dictionary of YAML files to help build the thing automatically 

 

For a bit of fun, a typical MSWord CV ninja2 template page, that takes care of the Word XML stuff behind the scenes, might look like this
