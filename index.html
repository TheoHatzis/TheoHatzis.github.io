This is where I keep my online CVs built from my repos on GitHUB.  

 

  • Short, HTML
  • Full CV MSWord, PDF

 

I like using Python Word automation and with the help of Ninja2 docxtpl templates,  some markdown and YAML files to help build the thing automatically  

For a bit of fun, a typical MSWord CV template in ninja2, that takes care of the Word XML parts behind the scene, might look like this
